    Indice dei Moduli¶

    Modulo Parametri discussi
    Script Manager navigazione pagine script, avvio/arresto script, installazione script Lua
    Quick Tags assegnazione tag per categoria, numero di campi (configurabile fino a 9), scorciatoie da tastiera (KP1-KP9)
    Tagging visualizzazione gerarchica ad albero, modifica percorso (change path), rinomina manuale tag
    Tag Collection ordinamento e valutazione in blocco, filtro per categoria
    Rename Tags nuovo nome tag, vecchio nome tag, esecuzione unificazione
    Lighttable organizzazione libreria, visualizzazione metadati
    Tag assegnazione etichette, sincronizzazione
    Tag rapidi filtro per tag, accesso veloce alle etichette

    Consigli e Avvertimenti¶
